# Variable Neighborhood Search (VNS) for TSP

## Overview
This project implements the **Variable Neighborhood Search (VNS)** metaheuristic to solve the **Traveling Salesman Problem (TSP)**.  
It includes:
- Python implementation of VNS
- Visualization of tours
- Example with 10 cities

## Features
- Shake and Local Search operators: Swap, Insert, Reverse
- Distance matrix calculation
- Plotting initial and final tours with costs
- Cost displayed with 2 decimal points

## Requirements
- Python 3.11+
- pandas
- numpy
- matplotlib
